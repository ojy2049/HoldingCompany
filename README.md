# HoldingCompany   

## Cobro (중개인용 애플리케이션)
중개사무실을 거점으로 운영하는 일반 공인중개사에 비해  
상대적으로 수익활동 지역이 자유로운 공동중개인이 서울시 구내의   
동별 중개인 평균 기대 소득과 예측 소득을 데이터로 제공 받을 수 있는 서비스 설계 및 구현

#### [개발환경]
> OS : Windows 10   
> 개발 도구 : 안드로이드 스튜디오, PyCharm   
> 사용 언어 : Kotlin, JAVA, Python   
> DB : AWS RDS(MySQL)
